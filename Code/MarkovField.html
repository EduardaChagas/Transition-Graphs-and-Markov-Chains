Generate autoregressive sequencies

Here, we evaluated two types of sequences:

  • 1000 elements of Autoregressive sequence
  • 1000 elements of Autoregressive sequence + 40 synthetic elements generate by Markov Transition Ordinal Graphs

The synthetic elements were obtained using the mean + standard deviation of the ordinal pattern chosen by the Markov transition graph:

AR <- function(n, phi, y0){
  x = c(y0, rep(0,n))
  for(i in c(2:(n+1))){
    error = rnorm(1,mean = 0, sd = 1)
    x[i] = (phi * x[i-1]) + error
  }
  return(x[2:(n+1)])
}

set.seed(456)
